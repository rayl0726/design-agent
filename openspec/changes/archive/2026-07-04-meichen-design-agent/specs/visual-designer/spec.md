## 新增需求

### 需求：生成概念效果图
系统须生成 2-3 张 AI 概念效果图（透视图），展示提案装置在实际场地中的效果。

#### 场景：基于照片上下文生成效果图
- **当** 用户已上传现场照片且 L1 概念已确认
- **则** 视觉设计 Agent 构建英文图像生成 Prompt，包含：设计概念描述、氛围关键词、照片分析得到的空间上下文、风格修饰词（如"architectural visualization"、"soft lighting"）
- **且** 调用图像生成服务 2-3 次，使用略有差异的摄像机角度
- **且** 返回生成图片及元数据：使用的 Prompt、seed、suggested_view_angle

#### 场景：基于 CAD 上下文生成效果图
- **当** 用户已上传 CAD 但未上传照片
- **则** 系统使用 CAD 推导的空间尺寸指导 Prompt
- **且** 生成展示大致尺度和空间关系的效果图
- **且** 附加声明："效果图基于 CAD 尺寸，实际比例可能存在偏差"

### 需求：生成色彩材质板
系统须生成一张色彩材质板（色彩材质板），展示提案色板及材料样本。

#### 场景：色彩材质板生成
- **当** L1 氛围描述包含颜色关键词和材料建议
- **则** 系统提取或生成 4-6 个主色 hex 色值
- **且** 为每种颜色从知识库推荐 1-2 种匹配材料
- **且** 使用 Pillow 生成合成图：色块 + 材料缩略图占位 + 材料名称标签
- **且** 附带文本表格：color_hex、material_name、finish_type、application_area

### 需求：确保视觉一致性
系统须确保概念效果图和色彩材质板与已确认的 L1 概念保持视觉一致。

#### 场景：一致性检查
- **当** 视觉设计 Agent 生成 L2 交付物
- **则** 系统调用 LLM 对比 L2 图像 Prompt 与 L1 概念故事及氛围
- **且** 标记不一致（如 L1 写"清新明亮"但渲染 Prompt 偏暗调）
- **且** 检测到不一致时，在图像生成前自动修正 Prompt
